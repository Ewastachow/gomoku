start :: Int -> Int -> (Int, Int) -> IO ()

Receive the game arguments and run the game.

play starts up a graphics window and sets up handlers for dealing with inputs and updating the world state.

drawWorld converts the world state into a gloss Picture

handleInput is called whenever there is an input event, and if it is a human player's turn should update the board with the move indicated by the event

updateWorld is called 10 times per second (that's the "10" parameter) and, if it is an AI's turn, should update the board with an AI generated move
